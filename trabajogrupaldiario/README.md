# Portal de Noticias - Portada

Este es un proyecto para la creación de la portada de un portal de noticias (diario), desarrollado como trabajo práctico grupal.
Fue diseñado con el enfoque **Mobile First**, utilizando HTML5 semántico y CSS **Flexbox**. Todo el CSS está modularizado y unificado usando la regla `@import`.

## Características
- Diseño propio responsivo.
- Menú "hamburguesa" para versiones móviles y barra horizontal para escritorio.
- Componentes modulares (`header`, `nav`, `news`, `ads`, `footer`).
- Uso de `normalize.css` para consistencia.
